Classes vs. Ids

Quando devemos utilizar as classes?

Aqui estao duas regras:

  • Quando quisermos aplicar o mesmo estilo a multiplos elementos
  • Mas so a elementos com o mesmo nome

Uma classe que pode ser utlizada em multiplos elementos e um elemento pode ter multiplas classes



Quando devemos utilizar Ids?

Aqui estao duas regras:

    Quando nao precisamos de aplicar o mesmo estilo a multiplos elementos
  • Para ter um selector unico a partir de um elemento comum

Um ID nao deve ser utilizado em multiplos elementos e um elemento nao deve ter multiplos IDs
